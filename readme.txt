A simple plugin to add social share buttons to your posts.

== Description ==

This plugin requires the [Pluton](https://plutonwp.com/) theme to be installed.

== Installation ==

1. Upload `pluton-social-sharing` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!

== Frequently Asked Questions ==

= I installed the plugin but the social share buttons is not in my post =

This plugin will only work with the [Pluton](https://plutonwp.com/) theme.
